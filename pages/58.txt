   <h1 id="firstHeading" class="firstHeading mw-first-heading">Стилизация</h1>

     Материал из Википедии — свободной энциклопедии

    <div id="mw-content-text" class="mw-body-content mw-content-ltr" lang="ru" dir="ltr">
     <div class="mw-parser-output">
      <ul>
       <li><b><a href="/wiki/%D0%A1%D1%82%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F_(%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE)" title="Стилизация (изобразительное искусство)">Стилизация (изобразительное искусство)</a></b> — воспроизведение образной системы и формальных особенностей одного из стилей прошлого, использованных в новом художественном контексте; творческий метод, основанный на отношениях: стилизуемый образец&nbsp;— стилизующая <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D0%BE%D0%B7%D0%B8%D1%86%D0%B8%D1%8F_(%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE)" title="Композиция (изобразительное искусство)">композиция</a>.</li>
      </ul> 
      <ul>
       <li><b>Стилизация в литературе</b>&nbsp;— близка таким понятиям, как <a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%80%D0%B0%D0%B6%D0%B0%D0%BD%D0%B8%D0%B5" title="Подражание">подражание</a>, эпигонство, <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%BE%D0%B4%D0%B8%D1%8F" title="Пародия">пародия</a>, <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%B4%D0%B8%D1%86%D0%B8%D1%8F" title="Традиция">традиция</a>, влияние, но существенно отличается от них:</li> 
       <li>Традиция предполагает прогрессивное развитие искусства, при котором творчество гениев оказывает влияние на мастеров «второго ряда» (<a href="/wiki/%D0%9B%D0%B8%D1%82%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%BD%D0%B0%D1%8F_%D1%8D%D0%BD%D1%86%D0%B8%D0%BA%D0%BB%D0%BE%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F_(1929%E2%80%941939)" class="mw-redirect" title="Литературная энциклопедия (1929—1939)">«Литературная энциклопедия»</a> приводит в качестве примеров влияние <a href="/wiki/%D0%9F%D1%83%D1%88%D0%BA%D0%B8%D0%BD" class="mw-redirect" title="Пушкин">Пушкина</a> на <a href="/wiki/%D0%9B%D0%B5%D1%80%D0%BC%D0%BE%D0%BD%D1%82%D0%BE%D0%B2" class="mw-redirect" title="Лермонтов">Лермонтова</a>, <a href="/wiki/%D0%A4%D0%B5%D1%82" class="mw-redirect" title="Фет">Фета</a> и <a href="/wiki/%D0%A2%D1%8E%D1%82%D1%87%D0%B5%D0%B2" class="mw-redirect" title="Тютчев">Тютчева</a> на <a href="/wiki/%D0%91%D0%BB%D0%BE%D0%BA,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Блок, Александр Александрович">Блока</a>). Стилизация часто подразумевает ретроспективный уход в прошлое. Термин не используется в отношении творчества крупных мастеров, демонстрирующих оригинальность и самобытность творческого метода, например, при описании <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%B7%D0%BA%D0%B8_%D0%9F%D1%83%D1%88%D0%BA%D0%B8%D0%BD%D0%B0" title="Сказки Пушкина">сказок Пушкина</a>;</li> 
       <li>Подражание подчиняет творчество эпигона художественному методу другого мастера, обычно современника или творившего в недалёком прошлом; при этом сходство с образцом не подчёркивается а, наоборот, скрывается;</li> 
       <li>Стилизация как <a href="/wiki/%D0%93%D1%80%D0%BE%D1%82%D0%B5%D1%81%D0%BA" title="Гротеск">гротеск</a> — пародия преувеличивает имитируемые особенности оригинала и даёт им комическую трактовку;</li> 
       <li>Стилизация в периоды <a href="/wiki/%D0%94%D0%B5%D0%BA%D0%B0%D0%B4%D0%B0%D0%BD%D1%81" class="mw-redirect" title="Декаданс">декаданса</a>. В русской литературе приёмы стилизации были ярко продемонстрированы мастерами <a href="/wiki/%D0%94%D0%B5%D0%BA%D0%B0%D0%B4%D0%B0%D0%BD%D1%81" class="mw-redirect" title="Декаданс">декаданса</a>: <a href="/wiki/%D0%9A%D1%83%D0%B7%D0%BC%D0%B8%D0%BD,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Кузмин, Михаил Алексеевич">Кузминым</a>, <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B8%D0%B7%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Ремизов, Алексей Михайлович">Ремизовым</a>, <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2,_%D0%92%D1%8F%D1%87%D0%B5%D1%81%D0%BB%D0%B0%D0%B2_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Иванов, Вячеслав Иванович">Вяч. Ивановым</a>, <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%8C%D0%BC%D0%BE%D0%BD%D1%82" class="mw-redirect" title="Бальмонт">Бальмонтом</a>.</li>
      </ul>
